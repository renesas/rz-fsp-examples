This is the Preceding Project running on the CM33 core. 
The Example Project of RZ/V2H running on the CR8 core requires the Preceding Project (or Smart Bundle file) 
for building or debugging.

Usage: Import this Project into your workspace and build it before importing the main Example Project running 
on the CR8 core.
